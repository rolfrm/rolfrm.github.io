In my efforts for creating effective 3D math in Lisp/javascript and came up with an interesting macro, mat4:multiply!.

(defun mat4:rotate-x (m rad)
  (const ((cosA (math:cos rad))
             (sinA (math:sin rad))
             (isinA (- 0 sinA)))
     (mat4:multiply! m 
                          1 0 0 0
                          0 cosA sinA 0
                          0 isinA cosA 0
                          0 0 0 1)))
